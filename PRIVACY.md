# GapGap Privacy Policy

Last updated: May 29, 2026

GapGap is a Chrome extension for local Chinese-English mixed typography proofreading.

## Data processed by the extension

GapGap processes text that you select on a webpage or type into the extension side panel. The extension may also store local settings, local proofreading history, and the latest proofreading result in Chrome storage on your device.

## How data is used

The text is used only to generate local typography corrections, display the proofreading result, support copy actions, and keep optional local history/settings.

## Data sharing

GapGap does not upload your selected or typed text to external servers. GapGap does not sell, transfer, or share user data with third parties.

## Remote code

GapGap does not load or execute remote JavaScript or WebAssembly. All extension code is included in the extension package.

## Contact

For privacy questions, contact the publisher through the Chrome Web Store listing or the GitHub repository.
